# A simple raytracer written in C
- first bounce 
- multithreaded using pthread library
- has texturmapping 
- basic blin-phong shading 

# Dependencies 
- This program needs devIL, SDL2 ,freetype2 and pthreads in order to compile successfully.

# how to build & run

## To build you:
```
make -j8;
```

## And to run we just: 
```
make run
```
